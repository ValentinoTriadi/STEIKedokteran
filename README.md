# Dependency

## Backend

1. Create virtual environment (optional)
2. ```sh
   cd app/api
   ```
3. ```sh
   pip install -r requirements.txt
   ```

## Frontend

1. ```sh
   npm i -g yarn
   ```
2. ```sh
   yarn
   ```

# HOW TO RUN

## Backend

1. ```sh
   cd app/api
   ```
2. ```sh
   python main.py
   ```

## Frontend

1. ```sh
   yarn
   ```
2. ```sh
   yarn run dev
   ```
3. open `localhost:3000`
